HTML5 Audio player. Works only with modern browsers; no flash fallback or silverlight fallback. For a full featured solution I suggest the fantastic MediaElementPlayer.js

The main goal of this project is to provide a solid approach to creating custom audio player representations, without enforcing an actual UI. The UI from a visual design level is left up to you to implement, though a fully functional implementation using Raphael.js can be found in the demos directory.

Note that this project is still under development, and may be broken at any given time.
