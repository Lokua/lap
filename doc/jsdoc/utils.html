(static) sliceRel(url, preSlash, trailingSlash)

Extracts final relative part of url, optionally keeping forward, backward, or both slashes. By default both front and trailing slashes are removed
Parameters:
Name Type Description
url String the url or filepath
preSlash Boolean keeps slash before relative part if true
trailingSlash Boolean keeps last slash after relative part if true
Source:
  • utils.js, line 92
Example
var url = 'http://momentsound.com/nreleases/m01_garo/';
u.sliceRel(url);               //=> m01_garo
u.sliceRel(url, false, false); //=> m01_garo
u.sliceRel(url, false, true);  //=> m01_garo/
u.sliceRel(url, true, true);   //=> /m01_garo/
u.sliceRel(url, true, false);  //=> /m01_garo

// sliceRel does not add a trailing slash if it wasn't
// there to begin with
url = 'http://momentsound.com/nreleases/m01_garo';
u.sliceRel(url, false, true); //=> m01_garo
u.sliceRel(url, true, true);  //=> /m01_garo
